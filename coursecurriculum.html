Course Curriculum

Welcome to the Web Development course. In this course, you will learn HTML, CSS, and JavaScript to make web pages.

About the Course

This course is for beginners and covers both theory and practical topics. You will also learn how to use special symbols like ©, ®, and <html>.

Modules

  1. HTML Basics
    • Introduction to HTML
    • HTML Tags
    • Links and Images
  2. CSS Styling
    • Colors and Backgrounds
    • Fonts and Text
    • Box Model
  3. JavaScript
    • Variables
    • Functions
    • Events

Course Objectives

  • Learn how websites work
  • Write simple HTML and CSS code
  • Make web pages interactive with JavaScript
